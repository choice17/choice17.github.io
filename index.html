Face Recognition

Face Recognition PCA MachineLearning Matlab Face detection

May 2017 - Sep 2017

Real-Time Face Recognition System

Face Recognition system able to achieve 95% facial identification accuracy on the evaluation data set. It implements EigenFace PCA methods, PCA methods depend on the variation of data set. The base system data is included with Extended Yale B face sets. The dataset includes variations of head pose and illumination, those variations can be extracted by PCA methods to compose pose features with high PCA coefficients. Whenever new candidate is registered, the system will retrain the SVM-classifier with the Yale dataset with newly captured pictures for new candidate. To achieve hard real-time performance with high accuracy of face detection, high filter threshold in traditional Vialo Jones face detection algorithm is used with KLT face tracking.

RPI3B Linux C Python ROS Dlib OpenCV

Oct 2017 – Dec 2017

To study ROS system, in this project, face detection module from OpenCV with facial landmarks detection from dlib are used. Vehicle On-road Dynamic Scenario Classification

ADAS MachineLearning DataAnalysis Matlab FeatureEngineer TimeSeriesAnalysis

Sep 2017 – Dec 2017

To understand driver behavior and vehicles on-road dynamic scenario, system can have a high abstract level of description on the road. It is observed that there is infinitely many variations of each scenarios, we defined the period of each event by average time duration of events in our label data. To build up a classification model on real-time on-road vehicle, this project tried to analysis vehicle on-board-diagnosis (OBD) data, with physiological signal. The neural network classifier able to achieve 70% classification accuracy on 5 vehicle maneuvers Project includes developing labeling GUI, data labeling. data analysis, feature selection, implement stratified cross validation and model optimization.

PID motor control on STM32F407-Discovery board in RTOS

STM32 C PID MotorControl IR-Sensor Sensor-Calibration LPF RTOS

Sep 2017 – Oct 2017

This is a class project, it is to build a PID system to control the distance between the embedded car and the front obstacle. The system connects with a IR sensor to detect the distance to obstacle, a motor driver to control PWM, motor speed and direction in RTOS uCOS-II system. In the project, we studied the sensor calibration method, data filtering, RTOS, STM32F4 series board interface, PID implementation. Whole project is writed in C, using µVision IDE - Keil 5.
